# billiard-pygame

# 🎱 Бильярд для двоих на Pygame

Классический бильярд с реалистичной физикой, рассчитанный на двух игроков за одним компьютером.

## 🎮 Управление

| Действие | Управление |
|----------|------------|
| **Игрок 1 (белый шар)** | Зажать **ЛКМ** на шаре → потянуть → отпустить |
| **Игрок 2 (жёлтый шар)** | Зажать **ПКМ** на шаре → потянуть → отпустить |
| Новая игра | **ПРОБЕЛ** |
| Сброс удара | **R** |
| Выход | **ESC** |

## ✨ Особенности

- 🎯 **Реалистичная физика** — упругие столкновения, трение, отскоки от бортов
- 🕹 **Два игрока на одном ПК** — поочерёдные удары
- 🏆 **Победа до 7 очков** — кто первый забьёт 7 цветных шаров
- ⚡ **Система силы удара** — чем дальше тянешь, тем сильнее удар
- 🎨 **Красивый визуал** — тени, блики, анимация удара, золотая подсветка
- 📐 **Полная разметка стола** — лузы, борта, сукно
